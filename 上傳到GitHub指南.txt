【🚀 上傳步驟】

1. 在 GitHub 創建新的 Repository
   - 名稱：nadu 或 nadu-official-store
   - 設為 Public（公開）
   - 不要勾選 "Initialize with README"（我們已經有了）

2. 在終端機執行：

   cd /Applications/XAMPP/xamppfiles/htdocs/nadu
   
   # 初始化 Git（如果還沒有）
   git init
   
   # 添加所有文件
   git add .
   
   # 提交
   git commit -m "Initial commit: Nadu Official Store - 完整電商網站系統"
   
   # 連接到 GitHub（替換成您的 URL）
   git remote add origin https://github.com/你的用戶名/nadu.git
   
   # 推送
   git branch -M main
   git push -u origin main

3. 在 GitHub Repository 設定：
   - About > Description: "Nadu Official Store - 功能完整的電商網站系統"
   - Topics: nextjs, react, typescript, php, mysql, ecommerce, tailwindcss
